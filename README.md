# NO_TELEMETRY

For information on the NO_TELEMETRY standard, see <https://no-telemetry.org/>.

## See also

- <https://opentelemetry.io/docs/specs/otel/configuration/sdk-environment-variables/#general-sdk-configuration>

## Examples

```sh
export ASTRO_TELEMETRY_DISABLED=1
export DO_NOT_TRACK=1
export QW_DISABLE_TELEMETRY=1
export LIGHTPANDA_DISABLE_TELEMETRY=true
export MEILI_NO_ANALYTICS=true
export SCARF_ANALYTICS=false
export TS_NO_LOGS_NO_SUPPORT=true
export SENTRY_DSN=""
export NEXT_TELEMETRY_DISABLED=1
export CLAUDE_CODE_ENABLE_TELEMETRY=0
export APOLLO_TELEMETRY_DISABLED=true
export GH_TELEMETRY=false
export DOTNET_CLI_TELEMETRY_OPTOUT=
export SAM_CLI_TELEMETRY=0
export AZURE_CORE_COLLECT_TELEMETRY=0
export GATSBY_TELEMETRY_DISABLED=1
export HOMEBREW_NO_ANALYTICS=1
export STNOUPGRADE=1
go telemetry off
gcloud config set disable_usage_reporting true
netlify --telemetry-disable
```

----

*Inspired by <https://no-color.org/>.*

See <https://donottrack.sh/>, a standard for respecting user privacy in software.
